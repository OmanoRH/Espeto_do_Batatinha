Fundamentos da Programação em Python

Estruturas de Seleção (if / elif / else)

As estruturas de seleção permitem que o programa tome decisões com base em condições. Em Python, usamos if, elif (senão se) e else (senão). 


          idade = int(input("Digite sua idade: "))
          if idade >= 18:
              print("Você é maior de idade.")
          else:
              print("Você é menor de idade.")

Estruturas de Repetição (for / while)

Utilizadas para repetir blocos de código. O for é usado em sequências e o while enquanto uma condição for verdadeira. 


          # Exemplo com for
          for i in range(5):
              print("Número:", i)
          
          # Exemplo com while
          contador = 0
          while contador < 5:
              print("Contador:", contador)
              contador += 1

Vetores e Matrizes (Listas e Listas Aninhadas)

Listas armazenam vários valores em uma única variável. Matrizes são listas dentro de listas. 


          # Lista (vetor)
          frutas = ["maçã", "banana", "laranja"]
          print(frutas[1])  # Saída: banana
          
          # Lista aninhada (matriz)
          notas = [[7, 8, 9], [6, 7, 10]]
          print(notas[1][2])  # Saída: 10

Funções e Procedimentos

Funções são blocos de código reutilizáveis. Elas podem retornar valores (return) ou apenas executar ações. 


          # Função com retorno
          def somar(a, b):
              return a + b
          
          resultado = somar(3, 4)
          print("Soma:", resultado)
          
          # Procedimento (sem retorno)
          def saudacao(nome):
              print(f"Olá, {nome}!")
          
          saudacao("Lucas")

Tratamento de Exceções (try / except)

Trata erros que podem ocorrer durante a execução do programa, evitando falhas inesperadas. 


          try:
              numero = int(input("Digite um número inteiro: "))
              resultado = 10 / numero
              print("Resultado:", resultado)
          except ZeroDivisionError:
              print("Erro: Divisão por zero não é permitida.")
          except ValueError:
              print("Erro: Você deve digitar um número inteiro.")
