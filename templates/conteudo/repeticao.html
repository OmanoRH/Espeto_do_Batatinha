Estruturas de Repetição (for / while)

Usadas para repetir blocos de código. O for é usado com sequências, e o while com condições lógicas.


# Usando for
for i in range(5):
    print("Contando:", i)

# Usando while
contador = 0
while contador < 5:
    print("Contador:", contador)
    contador += 1
