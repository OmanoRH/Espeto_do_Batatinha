Estruturas de Seleção (if / elif / else)

Permitem executar blocos diferentes de código com base em condições. Muito usadas para tomada de decisões.


idade = int(input("Digite sua idade: "))
if idade >= 18:
    print("Você é maior de idade.")
elif idade == 17:
    print("Você fará 18 em breve!")
else:
    print("Você é menor de idade.")
